#  Alumni Club – Monorepo

Ovo je monorepo projekat za **Alumni Club**, koji sadrži frontend i backend deo aplikacije.

##  Struktura projekta

Alumni-Club/
│
├── frontend/ # React + TypeScript (UI deo aplikacije)
├── backend/ # Node.js + Express + TypeScript (API server)
└── README.md # Ovaj fajl
